# Planck Choc One-Side Keymap

这是一个为 Planck 键盘设计的 ZMK 固件键位配置，采用分层布局设计，提供高效的输入体验。

## 特性

- **7个独立层级**：基础层、导航层、鼠标层、媒体层、数字层、符号层、功能层
- **智能修饰键**：在基础层使用 `mt` (mod-tap) 实现修饰键与字母键的组合
- **层切换键**：底行使用 `lt` (layer-tap) 实现按住切换层、点击发送按键
- **鼠标支持**：完整的鼠标移动、滚动和点击控制
- **媒体控制**：音量、播放控制和蓝牙设备切换
- **快捷操作**：剪切、复制、粘贴、撤销、重做等常用快捷键

## 层级说明

### 1. 基础层 (Base Layer)

标准的 QWERTY 布局，包含以下特点：
- 第一行：QWERTY 键位，左上角切换到符号层，右上角切换到导航层
- 第二行：使用 `mt` 实现修饰键与字母键组合 (A/S/D/F 对应 LGUI/LALT/LCTRL/LSHFT)
- 第三行：ZXCVB... 标准底排字母键
- 第四行：层切换键 (ESC/SPACE/TAB/RET/BSPC/DEL)

**层切换快捷键**：
- `ESC` (按住) → 第3层 (数字层)
- `SPACE` (按住) → 第1层 (导航层)
- `TAB` (按住) → 第2层 (鼠标层)
- `RET` (按住) → 第5层 (符号层)
- `BSPC` (按住) → 第4层 (媒体层)
- `DEL` (按住) → 第6层 (功能层)

### 2. 导航层 (Nav Layer)

提供方向键和编辑功能：
- 右侧：方向键 (←↓↑→)、Home/End、Page Up/Down、Insert
- 顶部右侧：撤销/重做、剪切/复制/粘贴

### 3. 鼠标层 (Mouse Layer)

完整的鼠标控制：
- 移动：鼠标指针四向移动
- 滚动：鼠标滚轮四向滚动
- 点击：左键、中键、右键点击

### 4. 媒体层 (Media Layer)

多媒体控制和蓝牙管理：
- 播放控制：上一首、下一首、播放/暂停
- 音量控制：音量减、音量增、静音
- 蓝牙设备：BT_SEL 0/1/2/3 选择设备，BT_CLR 清除配对
- 输出切换：OUT_TOG 切换输出设备

### 5. 数字层 (Num Layer)

数字小键盘布局：
- 数字：1-9、0
- 运算符：-、=
- 符号：[ ]、;、`、\、.

### 6. 符号层 (Sym Layer)

常用符号输入：
- 运算符：! @ # $ % ^ & * ( ) - + = [ ] { } | \ : ; < > ? .
- 括号：圆括号、方括号、尖括号

### 7. 功能层 (Fun Layer)

F键和系统功能：
- F1-F12：完整的功能键
- 系统键：Print Screen、Scroll Lock、Pause/Break
- 应用键：K_APP (菜单键)

## 快捷键参考

### 层级切换
- **左上角** → 符号层
- **右上角** → 导航层

### 常用快捷键
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+C` | 复制 |
| `Ctrl+V` | 粘贴 |
| `Ctrl+X` | 剪切 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |

## 使用说明

1. **日常输入**：在基础层进行常规打字
2. **方向导航**：按住 `SPACE` 进入导航层，使用右侧方向键
3. **鼠标操作**：按住 `TAB` 进入鼠标层，控制鼠标指针和滚动
4. **数字输入**：按住 `ESC` 进入数字层进行数字输入
5. **符号输入**：按住 `RET` 进入符号层输入特殊符号
6. **功能键**：按住 `BSPC` 进入媒体层控制播放
7. **F键使用**：按住 `DEL` 进入功能层使用 F1-F12

## 硬件支持

- 键盘：Planck (Choc v1 单侧版本)
- 固件：ZMK
- 指向设备：支持鼠标移动和滚动

## 配置文件

- `planck.keymap` - 主键位映射文件
- 使用 ZMK Devicetree 语法
- 包含 7 个完整的层级定义

## 自定义

如需自定义键位，请编辑 `planck.keymap` 文件：

1. 修改基础层字母或修饰键位置
2. 调整各层的功能键位
3. 添加或移除层切换键
4. 修改媒体控制或蓝牙设置

## 固件编译

```bash
# 编译固件
west build -b planck_rev6 -- -DSHIELD=planck_choc_one_side

# 刷写固件
west flash
```
